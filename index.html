This is the dataset we will work with. It contains all the anime on https://www.anime-planet.com/, a popular anime database. It lists the rank (by rating), English name, Japanese name, Type, Episodes, Studio, Release season, Genre tags, release year, end year, description, content warning, related manga, related anime, voice actors, and staff. We will use this information to draw results to common questions an anime enthusiast may have.

Which genre has the highest average rating?¶

genres_split = anime_data['Tags'].str.split(',').explode()

# Creating a DataFrame with the split genres
genres_df = pd.DataFrame({'Anime_ID': genres_split.index, 'Genre': genres_split.values})

# Merge the original DataFrame with the split genres DataFrame
anime_data_with_genres = pd.merge(anime_data, genres_df, left_index=True, right_on='Anime_ID')

# Now, 'Genre' is a separate column in the DataFrame

What is the most prefered episode count for viewers?¶

In [128]:
# Create a new column 'Episode_Category' based on the specified ranges
bins = [2, 15, 30, 50, 100, np.inf]
labels = ['2-15 Episodes', '16-30 Episodes', '31-50 Episodes', '51-100 Episodes', '100+ Episodes']
anime_data['Episode_Category'] = pd.cut(anime_data['Episodes'], bins=bins, labels=labels, right=False)

# Calculate the average rating for each episode category
average_rating_by_episode_category = anime_data.groupby('Episode_Category')['Rating'].mean().reset_index(name='Average_Rating')

# Visualize the average ratings for each episode category
plt.figure(figsize=(12, 6))
sns.barplot(x='Average_Rating', y='Episode_Category', data=average_rating_by_episode_category, palette='viridis')
plt.title('Average Rating of Anime by Episode Number')
plt.xlabel('Average Rating')
plt.ylabel('Episode Number')
plt.show()

There doesn't seem to be much conclusions we can draw from here since they are all so close, but one conclusion we can draw is that longer episode counts do not show a decrease in rating. It seems users are not drawn away by longer series.
